# mobile-app

This repository has been converted to a minimal Expo starter template.

What was done

- Shop-related components and cart context were removed to simplify the project.
- `App.js` restored to a plain Expo starter.

Run the app

1. Install dependencies (only if you haven't already):

```bash
npm install
```

2. Start Expo:

```bash
npm start
# or
npm run android
npm run ios
npm run web
```

Notes

- Expo packages and node modules were not installed or removed by this conversion — do not run global installs unless needed.
- Empty `component/`, `context/`, and `screens/` folders may remain; you can remove them if you prefer.
